【游戏名称】 台球游戏

【游戏介绍】 这是一款本地双击即可游玩的双人台球小游戏。你可以与朋友轮流击球，按照规则完成进球并争取胜利。

【玩法说明】 游戏为双人轮流制，玩家通过拖拽瞄准并释放击球。若首次撞击未命中合法目标球，将判为犯规并给予对手自由球。

规则要点：  
1. 合法目标球：未分球时可击任意非 8 号球；分球后只能击本方球。  
2. 首次撞击错误：首次撞击不合法目标球即犯规。  
3. 母球入袋：犯规，对手获得自由球。  
4. 8 号球：本方清台后击入可获胜，违规击入则对手获胜。  

操作：你可以通过鼠标/手指拖拽来瞄准并释放击球；获得自由球时，点击球台可放置母球。

目标：先清完本方目标球（实球/花球），再击入 8 号球即可获胜。

Tips：犯规会提示原因并交换回合；自由球可在球台内任意位置放置母球。

【技术介绍】 使用 Cocos Creator 3.8.8 构建的 H5 游戏，JavaScript 编写，Canvas 2D 渲染技术，支持 PC 和移动端浏览器。
